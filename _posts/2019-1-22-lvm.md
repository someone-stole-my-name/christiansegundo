## Identify the new disk
  
```
lsblk
```

## Initialize the disk

```
pvcreate /dev/vdh
```

## Add the PV to the existing VG
  
```
vgextend influxdb /dev/vdh
```
  
## Resize partition and filesystem

```
lvextend -l +100%FREE /dev/influxdb/influxdb
resize2fs /dev/influxdb/influxdb
```
